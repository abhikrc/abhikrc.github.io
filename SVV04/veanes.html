Invited  Talk

A Game approach to Software Testing
 

Margus Veanes (Microsoft Research, USA)

We give a brief overview of the model-based testing tool SpecExplorer developed at Microsoft Research and the modeling language Spec# used by SpecExplorer. The tool supports nondeterminism and provides a view of testing a nondeterministic system as a two player game. Generated tests are considered as game strategies. The underlying conformance relation is given by alternating refinement of interface automata.
